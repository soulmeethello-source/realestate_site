# 지역 규제 단타지수 거래건수
유동성		 중위가 중위평형 법인매수% 중위건축
단타지수 = 유동성(거래건수)과 법인매수 비중의 백분위를 반반 섞은 참고 지표(0~100)입니다. 거래 20건 이상 지역만 산출돼요.
읽는 법 — 유동성↑(빨리 팔림) + 법인매수%↑(매매사업자·단타 활발) = 단타 활발 신호. 단, 가격·입지·물건 상태는 별도 확인 필요. 법인매수가 높아도 고가 개발지역(예: 성수동)은 단타와 성격이 달라요.
